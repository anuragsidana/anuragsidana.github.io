Implementation Approach and Souce code Organization


  • I tried to add advanced functinalities to the game which were given us as an assignment. Hence the starting steps were same as we were directed to do in the assignment pdf.
    That pdf helped me a lot in building my basics and implementing initial code. At first my game was having a basic player object and an enemy object , with player having the basic feature of shooting and killing the enemy.The score corresponding to the player and its remaing power(health) performance was being shown on the screen.Also the enemies could only come from the upward direction and the player is at bottom line stopping enemies from crossing the bottom line.
    This was the basic functionality.


  • Now after this first of all I tried to optimize my code in the object oriented way (Hence my current code has been designed in oo design manner , hence easy to optimize ***)
    Now after this I added the following functionalities in my app:

    1. Player:(Player class overrides the Actor) I added two types of shoots in the player object : First , when the left button of mouse is pressed and 2nd when the right button is pressed.(This has been named as a special attack) I used sprite animations to show the movement of players and enemies so that it looks more real. Also the direction of shooting is being controlled by the mouse.

    2. Enemy:(Enemy Class overrides the parent class Actor) I also gave the enemy a power to shoot, the bullets are being fired by enemies at regular intervals in the direction of player. One more intersting thing,Now the enemies can come from any direction and they will always keep following the player.( :P :P ) And the bullets thrown by the enemies is that it disappears after a time span defined by me inside the code.

    3. Update: (class for two types of update objects) I created a new class which is responsible for creating two types of health objects in the game. i.e Two healths (+ sign in game) are being generated in the game at random places. First Type(Yellow): When user collides with yellow health health its power increases by an amount of 4. Second Type(Purple): When user collides with this health, its attacking speed increases by a considerable amount.

    4. Map:It basically represents the background of the game. It is an imgage, but a player cant go to certain places i.e (Stones , house, doors etc). Used map collision for the same.

    5. And I have commented out code at various points to have better understanding of my logic.
